The purpose of this assignment is fourfold
• To familiarize students with Advanced User Interface features and Design Patterns i.e.
Fragments, ToolBar, Menus, Preference Fragments
• To familiarize students with the Android Notification mechanism
• To familiarize students with basic data storage e.g. shared preferences
• To prepare students for their mobile application project
This assignment serves to bring together the different skills and knowledge obtained from previous
assignments while also testing new features, and your research process. The application that you
will develop is a Prescription Ordering Application.

I used the following environment to develop this assignment. Minimum SDK 23, Android Studio 2.3.3 and my SDK installs were 26.0.1
